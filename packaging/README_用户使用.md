# GeneSnap Workbench 0.3.4 使用说明

## 安装

双击 `GeneSnapWorkbench_Setup_v0.3.4.exe`，按提示完成安装。安装程序默认安装到当前 Windows 用户目录，不需要管理员权限，并创建开始菜单入口和桌面快捷方式。

## 第一次启动

第一次启动会要求选择“项目保存位置”。这里保存项目文件夹、设计报告、订购表、送测表和分析报告，不再默认藏在 C 盘 AppData。以后可以通过工具栏的“保存位置”修改默认目录；每次点击“新建项目”时也可以临时选择另一个位置。

## 新建项目

点击工具栏“新建项目”，先选择：

- shRNA 敲低。
- 表达类。
- GL002 报告载体。
- 全基因合成。

选择后才进入对应设计页，不会再默认跳到 shRNA。

项目录入内容较多时，使用窗口右侧的纵向滚动条查看后续字段；底部操作按钮会固定保留。项目总览表底部的横向滚动条用于查看“设计摘要”“可用克隆”等后续列。

接收日期和标准完工日期均按 `年-月-日` 显示。软件会按工作日自动建议完工日期，并禁止保存早于接收日期的完工日期。已有项目需要改期时，点击“修正完工日期”，选择新日期并填写原因；原日期和修正记录都会保留。

## 导入载体和供应商模板

- 表达和 reporter 载体入口支持 SnapGene `.dna` 和 GenBank `.gb/.gbk`。默认选择左右酶切位点并自动计算同源臂；也可以切换为“手动粘贴同源臂”。软件会显示文件格式、长度、拓扑、切点和同源臂预览，不再要求填写 0-based 边界。
- 供应商 Excel 模板会先自动识别表头和客户信息。映射页显示 `B · 引物名称`、`C · 引物序列` 这类下拉项，不需要手工填写 Excel 列号。
- 引物订购表可自动写入订购日期、Gene ID、NM 号和对应 PCR/oligo 产物长度；合并单元格会自动写入可编辑的左上角单元格。
- 自动识别正确时直接确认保存；不需要的可选字段选择“不填写”，不会因此报错。
- 模板保存后可在新项目中直接选择，不需要再次映射。

## 项目详情页

“项目”“oligo”“实验”“历史”是软件生成的只读记录，可以选择单元格后按 `Ctrl+C` 复制。实验状态通过上方操作按钮推进，测序判定由“分析测序”写入；“文件”页可双击打开文件，也可复制路径。

项目总览分别显示内部编号、引物送单号、引物订单号、测序送样号和测序订单号。标记引物已订购时三项引物阶段编号都可以留空，之后可填写修改说明再补录。项目完成后这些编号保持只读。

进入质粒抽提前，软件只列出自动 PASS 或经人工确认可用的克隆。默认不代替用户勾选；每个 Target/构建至少选择一个，同一组允许多选。历史页中的事件、原状态和新状态均使用中文显示。

主项目表和右侧详情之间的分隔线可以拖动。操作按钮较多时使用动作区下方的横向滚动条；鼠标悬停在下拉框、数字框或日期框上滚轮不会再改变数值。

## shRNA 自动设计

1. 输入项目号、基因和物种。
2. 输入 RefSeq 转录本号后点击“查询 CDS”，也可以直接粘贴 CDS/FASTA。
3. 点击“Broad + NCBI BLAST 自动设计”。
4. 第一次使用需要阅读并确认 Broad GPP 当前服务条款。
5. 软件依次执行 Broad GPP 候选评分、`>100 bp` 首轮间距选择、NCBI BLAST 脱靶筛选和 Broad Full oligo 校验。
6. 在线结果表会显示 target、Broad 得分、位置、BLAST 状态、脱靶说明和 oligo 来源。

NCBI BLAST 是公共共享服务，正常一次可能需要 1-2 分钟。软件会显示当前步骤，主界面不会假死。Broad 或 NCBI 临时不可用时，可以使用“离线候选（备用）”，但软件会明确标记为需要人工 BLAST，不会把未验证候选显示成自动通过。

## 使用边界

- 软件自动通过不等于湿实验验证通过。
- Broad GPP 当前条款面向研究用途，商业使用可能需要另行取得许可。
- 实验室改造载体应导入真实图谱并确认 protocol，不要用名称相似的公开图谱替代。
