# 更新记录

本项目使用语义化版本号。当前仍处于 Beta 验证阶段。

## 0.3.4 - 2026-07-14

### 新增与改进

- 表达载体导入支持按酶切位点自动定位同源臂，也支持手动粘贴 F/R 同源臂。
- 引物订购模板新增订购日期、Gene ID、转录本号和扩增产物长度映射。
- 项目详情分层记录引物送单号、引物订单号、测序送样号和测序订单号，并保留修改历史。
- 质粒抽提改为从真实可用克隆中人工选择，支持同一构建选择多个克隆。

### 修复

- 修复供应商 Excel 合并单元格无法写入、擎科扩展产物长度列无法识别的问题。
- 修复项目面板滚动、分栏、历史中文显示和完成项目关键字段锁定问题。
- 保留 0.3.3 的 UTF-8 GenBank 图谱导出及英文 Windows 路径兼容修复。

### 验证

- 公开仓库全量源码测试通过：312 tests、96 subtests。
- Windows 便携版和安装器完成四类工作流烟雾测试。

## 0.3.3 - 2026-07-14

### 新增

- 为窗口、任务栏、EXE、安装器、卸载项和快捷方式统一加入应用图标。
- 建立公开仓库所需的许可证、第三方声明、贡献指南、安全说明和自动测试工作流。
- Windows 打包后自动运行四类工作流的成品冒烟测试，并随成品附带许可证文件。

### 安全与发布

- 清理示例中的个人信息和内部编号。
- 将随软件提供的公开载体资源整理为仅含序列与来源链接的记录。
- 修复英文 Windows 代码页下中文及 `Δ` GenBank 图谱无法导出的问题。
- 兼容 Windows 长路径与 8.3 短路径差异，并更新 GitHub Actions 运行环境。

## 0.3.2 - 2026-07-13

### 新增与整合

- 将 shRNA、表达类、promoter-luciferase、SYN 和项目管理整合到同一 Windows 桌面程序。
- 支持供应商 Excel 模板映射、订购信息档案、多轮送测和测序分析。
- 提供 Windows 安装版和便携版构建流程。

### 修复

- SnapGene `.dna` 文件按二进制模式读取。
- 模板映射使用带列字母和表头名称的选项，不再要求输入数字列号。
- 日期显示完整年份并禁止早于接收日期的完工日期。
- 项目录入窗口适配较小屏幕，项目总览保持可见滚动条。

### 限制

- CKO/Flox 尚未实现。
- 实验室改造载体需要用户自行导入并确认。
- 在线 shRNA 流程依赖 Broad GPP 和 NCBI 服务可用性。
